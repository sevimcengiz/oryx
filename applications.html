5.1 Oryx-MRSI GUI Tutorial

This tutorial describes each step of the Oryx-MRSI analysis workflow using the graphical user interface. We will learn how to start the GUI, select the spar file, select requirements information an the Main Gui Page and it will enables us to use 8 different modules.
